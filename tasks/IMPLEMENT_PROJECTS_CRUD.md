# Task: Implement CRUD Forms for Projects

## Objective
Create and implement forms to handle Create, Read, Update, and Delete (CRUD) operations for projects.

## Requirements
1. Implement a form to create new projects.
2. Add functionality to edit existing projects.
3. Implement project deletion with a confirmation step.
4. Ensure all forms have proper validation and error handling.
5. Update the UI to reflect changes immediately after CRUD operations.
6. You should the project service (`src/services/project.ts`) for all database operations. Add anything you may need.
